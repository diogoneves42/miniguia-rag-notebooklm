# Entrega

**Repositório:**  
`https://github.com/diogoneves42/miniguia-rag-notebooklm`

**Descrição:**  
Projeto de estudo sobre **Retrieval-Augmented Generation (RAG)** desenvolvido com apoio do NotebookLM. O material reúne contexto, objetivos, curadoria de fontes abertas, registro de prompts, dificuldades observadas durante a exploração e um miniguia final com resumos, glossário e prompts reutilizáveis.
